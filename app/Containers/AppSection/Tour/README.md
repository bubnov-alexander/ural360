# Tour

Контейнер хранит маршруты Ural360, перенесенные из WordPress `product`.

На первом этапе перенесены:

- Сплав по Чусовой: Усть-Утка - Ёква, 24 км;
- Сплав по Чусовой: Сулём - Ёква, 40 км;
- Маршрут «Индивидуальный»;
- Маршрут Усть-Утка - Верхняя Ослянка, 86 км.

Изображения маршрутов хранятся через Spatie Media Library в collection `tour_image`.
