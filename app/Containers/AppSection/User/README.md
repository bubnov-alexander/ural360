# User Container

Минимальный контейнер пользователя для входа в Filament.

На текущем этапе здесь остаются только:

- модель `User`;
- миграция таблицы `users` и `sessions`;
- seeder стартового администратора из `.env`.

API-логика пользователей, роли, permissions и отдельная пользовательская авторизация удалены.
