# Settings

Контейнер хранит редактируемые настройки проекта Ural360 через `spatie/laravel-settings`.

На первом этапе сюда перенесены общие данные из WordPress:

- название и URL сайта;
- телефон, email и email получателя заявок;
- адрес;
- ключ и embed-скрипт Яндекс.Карты.

Редактирование доступно в Filament: `Настройки -> Контакты и карта`.
