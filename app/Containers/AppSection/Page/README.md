# Page

Контейнер хранит базовые страницы сайта Ural360, перенесенные из WordPress.

Страницы разделены по `type`, чтобы фронт мог выбирать нужный шаблон:

- `home` — главная;
- `content` — обычная контентная страница;
- `contacts` — контакты;
- `routes` — страница популярных маршрутов;
- `services` — страница услуг;
- `legal` — юридические страницы.
